# Initial Candidate Libraries

This directory holds raw data for the primary screening stage (not tracked by Git).

## Small-Molecule Line

- `initial_compounds.smi` — space-separated: `SMILES  MOL_ID`
- Source: ChEMBL / ZINC20 subset, or built-in test set in notebooks

## Protein/Peptide Line

- `pd_l1_nanobody_seed.fasta` — antibody seed sequence (e.g. KN035)
- Mutation library generated inside notebooks

## Nucleic-Acid Line

- `CD274_mRNA.fasta` — human CD274 (PD-L1) mRNA (NCBI NM_014143, auto-downloaded in Module 1)
- `cd274_target_metadata.json` — target regions and siRNA design parameters
- `reference/chr22.fa` — hg38 chr22 off-target demo reference (auto-downloaded in Module 3; use full hg38 in production)
- siRNA candidates auto-generated by notebook sliding window
